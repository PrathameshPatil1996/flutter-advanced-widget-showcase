A **comprehensive Flutter demo application** showcasing advanced widgets, animations, UI designs, authentication techniques, and integrations such as Firebase notifications and Google ML Kit.

This project demonstrates multiple Flutter concepts in one application and is designed as a **learning and reference project for Flutter developers**.

---

# 📱 App Overview

The application contains multiple modules demonstrating real-world Flutter features including:

* Animations
* Sliver widgets
* Advanced scrolling
* Authentication methods
* Navigation transitions
* Lazy loading
* ML Kit integrations
* Notifications
* UI design patterns
* Chatbot interface
